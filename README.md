# LANGUAGE DETECTION AND TRANSLATION

This project is a Streamlit web application that translates spoken voice and written text between multiple languages. It supports real-time speech recognition, automatic language detection, text translation, and maintains a translation history for user reference.

# Project Overview

The application allows users to either speak or type text in one language and instantly translate it into another selected language. It leverages speech recognition for voice input and Google Translate APIs for accurate multilingual translation.

# Features

1. Voice-to-text translation using speech recognition

2. Text-to-text translation

3. Automatic source language detection

4. Supports multiple languages (English, Hindi, French, German, Spanish)

# Technologies Used

Python, Streamlit, SpeechRecognition, Googletrans, Langdetect

# Supported Functionalities

1. Convert spoken audio into text

2. Detect the input language automatically

3. Translate input into a selected target language

4. Display translated output instantly

5. Store and view past translations
